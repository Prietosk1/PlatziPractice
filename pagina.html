Comandos basicos de GIT

  • cd (directorio): se usa para navegar a traves de carpetas o directorio
  • init: iniciara un repositorio y el estado staged dentro del directorio seleccionado
  • mkdir: crear una carpeta dentro del directorio seleccionado
  • touch: crear un archivo de cualquier clase dentro del directorio seleccionado
  • rm: elimina archivos o documentos
  • branches: son ramas las cuales se usan para almacenar ciertos commits, ideal para trabajar de manera experimental
  • checkout: se utiliza para viajar a commits anteriores u otras ramas
